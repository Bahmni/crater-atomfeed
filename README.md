# Crater Atomfeed

#### Pre-requisite
* mvn 3.6.2. 
* Java 1.8, works on 11 as well (with some warnings) 

####  running the app 
> cd /workspace-dir/crater-atomfeed/

> cd /workspace-dir/crater-atomfeed/feed-integration-webapp/

> cd feed-integration-webapp/

> mvn clean install 

> mvn spring-boot:run
